Урок 1. Знакомство с HTML

HTML (HyperText Markup Language) и CSS (Cascading Style Sheets) являются основными технологиями для создания веб-страниц. HTML предоставляет структуру страницы, в то время как CSS занимается оформлением и дизайном. В этом уроке мы познакомимся с основами HTML и CSS.

Часть 1: Основы HTML

HTML документ состоит из ряда элементов, которые выглядят как теги. Теги обрамляются угловыми скобками. Например, `<html>`, `<head>`, `<body>`. Структура HTML-документа примерно следующая:

Основные HTML-теги
  • - `<h1>`, `<h2>`, ..., `<h6>`: заголовки разных уровней.
  • - `<p>`: абзац текста.
  • - `<a href="URL">`: ссылка.
  • - `<img src="URL" alt="описание">`: изображение.
  • - `<ul>`, `<ol>`, `<li>`: ненумерованные и нумерованные списки

Часть 2: Основы CSS

CSS используется для стилизации веб-страниц. С помощью CSS вы можете задавать цвета, шрифты, расстояния между элементами, и многое другое. CSS можно добавлять непосредственно в HTML-файлы, но чаще его помещают в отдельные файлы с расширением .css.
